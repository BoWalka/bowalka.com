
# Bowalka Cyberdecks - Premium Clamshell Mini Laptops

Futuristic cyberdeck-style laptops with neon glows, high-tech specs, running Arch/Ubuntu. Static demo site with cart and checkout.

## Features
- Cyberpunk matrix raining emoji background
- Responsive design (mobile-first)
- LocalStorage cart
- Fake checkout (demo)
- Unsplash images for products
- Glowing animations, CRT effect
