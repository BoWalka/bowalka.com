# Bowalka Designs - Official Website

Professional graphic design, video production, and AI prompt engineering services.

## Website Structure

This repository hosts the GitHub Pages website for [bowalka.com](https://bowalka.com)

### Pages

1. **index.html** - Landing Page
   - Image board showcasing services and projects
   - Featured work highlights
   - Quick links to services and quote request
   
2. **design.html** - About Page
   - Adobe After Effects & Illustrator expertise
   - Professional background and experience
   - Portfolio preview
   - Skills and specialties
   
3. **services.html** - Services Page
   - Detailed service offerings
   - Commercial video production
   - Social media advertising
   - Brand identity design
   - AI prompt engineering
   - Infographic & data visualization
   - Hype creation & campaigns
   
4. **quote.html** - Quote Request Form
   - Contact information collection
   - Project details and requirements
   - Budget and timeline selection
   - Custom quote submission

### Assets

- **matrix.js** - Animated matrix rain background effect with emojis
- **style.css** - Global styles and CSS variables
- **script.js** - Additional JavaScript functionality (if needed)

## Features

- 🎨 Dark-themed, modern design
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Fast loading with optimized assets
- 🎬 Matrix rain animation background
- 🤖 AI prompt engineering services highlighted
- 💼 Professional portfolio showcase
- 🔒 Privacy-first approach

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- GitHub Pages hosting
- FormSubmit for quote form handling
